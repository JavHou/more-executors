more-executors¶

A library of composable Python executors.

  • Documentation
  • Source
  • PyPI

This library is intended for use with the concurrent.futures module. It includes a collection of Executor implementations in order to extend the behavior of Future objects.

Compatible with Python 2.6, 2.7 and 3.x.

This documentation was built from more-executors 1.16.0.dev0, revision 34f9b1d.
